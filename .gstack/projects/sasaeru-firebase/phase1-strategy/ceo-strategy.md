# CEO戦略方針 — SASAERU Phase 1

## Phase 1 最優先KPI
**クラブ×企業マッチング成立件数: 月間3件（初月）→ 月間10件（3ヶ月後）**
登録数ではなく「実際につながった件数」が唯一の証明。

## Firebase選択の戦略的理由
| 観点 | 理由 |
|------|------|
| コスト | 無料枠でMVP検証。スケール後に課金移行 |
| 運用 | サーバーレス。NPO少人数でも継続運用可 |
| 保守 | 認証・DB・ストレージ一体 |

## Netlify公開の利点
- Gitプッシュ = 即デプロイ
- 無料SSL + CDN
- 環境変数でFirebaseキーを保護
