# SASAERU 公開環境 目視テストチェックシート

**バージョン:** 1.0  
**作成日:** 2026-04-16  
**テスト形式:** 公開URL上での手動・目視テスト  
**想定テスト時間:** 約90〜120分（全項目実施の場合）

---

## テスト環境・前提条件

### 必要なアカウント

| 種別 | 内容 | 備考 |
|------|------|------|
| 管理者アカウント | Firebase Auth登録済み + Custom Claims `admin: true` 付与済み | admin.html ログイン用 |
| クラブオーナーアカウント | Firebase Auth登録済み + Firestore `organizations` ドキュメント紐付き済み | mypage.html ログイン用 |
| 公開クラブデータ | `status: "public"` の団体が1件以上Firestoreに存在 | club.html 表示確認用 |

### テスト端末（推奨セット）

| デバイス区分 | 具体例 | 目的 |
|-----------|------|------|
| スマートフォン（iOS） | iPhone SE / iPhone 14 | モバイルUI確認 |
| スマートフォン（Android） | Pixel 7 / Galaxy S23 | クロスOS確認 |
| タブレット | iPad Air / Android 10インチ | 中間サイズ確認 |
| ノートPC | MacBook / Windows Laptop | デスクトップUI確認 |
| デスクトップ（広画面） | 1440px以上のモニター | 最大幅レイアウト確認 |

### ブラウザ（各端末で確認）

- Chrome（最新）
- Safari（iOS / macOS）
- Firefox
- Edge（Windows）

### テスト前の準備

- [ ] 公開URLをブラウザで開けることを確認
- [ ] 管理者アカウントのメールアドレス・パスワードを手元に用意
- [ ] クラブオーナーアカウントのメールアドレス・パスワードを手元に用意
- [ ] 別タブまたは別ブラウザで Firebase Console を開いておく（データ確認用）
- [ ] 開発者ツール（DevTools）でConsoleエラーが出ていないことを確認
