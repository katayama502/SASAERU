# SASAERU システム機能一覧

> 島根県の地域クラブチームと支援企業・個人をマッチングするプラットフォーム

---

## 👥 一般ユーザー（未ログイン）

| 機能 | 説明 |
|------|------|
| クラブチーム一覧表示 | カテゴリ・地域・キーワードで絞り込んで公開中の団体を検索できる |
| カテゴリタブ切り替え | スポーツ / 文化・芸術 / 福祉 など Firestore から動的に読み込まれるタブで絞り込める |
| カテゴリ別件数表示 | 各タブに該当件数をリアルタイムで表示する |
| クラブ詳細ページ閲覧 | 団体紹介・活動日記・支援メニューを閲覧できる |
| 支援申請送信 | 支援メニューから企業名・担当者・メッセージを入力して申請を送信できる |
| お問い合わせ送信 | 運営事務局へフォームから問い合わせを送信できる |
| クラブ登録申請 | 新規団体として登録申請ができる（審査待ち状態で登録） |
| メール確認 | 登録後に送られる確認メールのリンクをクリックして本人確認できる |

---

## 🏃 クラブチーム（ログイン済み）

### プロフィール管理

| 機能 | 説明 |
|------|------|
| 団体情報編集 | 団体名・カテゴリ・地域・紹介文・タグ・画像を編集できる |
| 地域複数選択 | 活動地域を複数選択（島根県の市区町村リストから選択）できる |
| クラブ紹介テンプレート | 紹介文のひな形をダウンロードまたはテキストエリアに一括反映できる |
| 再申請 | 審査却下後に修正して再申請できる |

### 支援メニュー管理

| 機能 | 説明 |
|------|------|
| メニュー作成・編集 | タイトル・支援種別・希望金額・説明・リターンを設定してメニューを作成できる |
| 公開 / 非公開切り替え | メニューの公開状態をワンクリックで切り替えられる |
| 表示順ドラッグ変更 | メニューカードをドラッグ&ドロップで並べ替えられる |
| 支援完了ボタン | マッチングが成立したことをボタンで記録し、継続表示 or 非表示を選択できる |
| マッチング実績バッジ | 成立件数をカードに「実績 N 件」として表示する |
| 非表示バッジ | 非表示設定中のメニューに「非表示中」バッジを表示する |
| メニュー削除 | 不要なメニューを削除できる |

### 活動日記

| 機能 | 説明 |
|------|------|
| 投稿作成・編集・削除 | 活動報告をタイトル・本文・画像付きで投稿できる |
| 投稿一覧表示 | 自団体の投稿を一覧で管理できる |

### 申請受信

| 機能 | 説明 |
|------|------|
| 支援申請一覧 | 自団体への支援申請を一覧で確認できる |
| 対応済みマーク | 申請を「対応済み」に更新できる |

---

## 🏢 管理者（admin ロール）

### ダッシュボード

| 機能 | 説明 |
|------|------|
| 概要カード表示 | 審査待ち件数・公開中団体数・申請件数をリアルタイムで確認できる |

### 団体管理

| 機能 | 説明 |
|------|------|
| 団体一覧・詳細閲覧 | 全ステータス（審査待ち・公開・却下）の団体を一覧・詳細で確認できる |
| 審査（公開 / 却下） | 申請中の団体を審査して公開または却下できる |
| 団体情報編集 | 任意のフィールドを管理者権限で修正できる |

### 支援申請管理

| 機能 | 説明 |
|------|------|
| 申請一覧（リスト / カンバン） | 全申請をリスト形式またはカンバンボードで管理できる |
| ステータス更新 | 申請を「未対応 → 対応済み」に更新できる |
| マッチング成立処理 | 申請に対し「継続支援あり / なし」を選択してマッチング成立にできる |
| CSV エクスポート | 申請データ（10 フィールド）を Excel 対応の CSV でダウンロードできる |

### 支援メニュー管理

| 機能 | 説明 |
|------|------|
| 全団体のメニュー一覧 | 全団体の支援メニューを matchingCount・isVisible 付きで確認できる |

### 管理者管理

| 機能 | 説明 |
|------|------|
| 管理者招待 | メールアドレスを指定して管理者権限を付与できる |
| 管理者一覧・権限削除 | 現在の管理者一覧を確認し、権限を削除できる |

## 🔄 自動処理（Netlify Scheduled Functions）

| 処理 | タイミング | 内容 |
|------|-----------|------|
| 週次リマインド送信 | 毎週月曜 9:00 JST | 支援申請から 7 日以上経過して未対応のままの団体へリマインドメールを送信。前回送信から 7 日経過するたびに繰り返す。支援完了で自動停止。 |

## 📊 データ管理

| コレクション | 説明 |
|-------------|------|
| `organizations` | 団体プロフィール（ステータス・カテゴリ・地域・画像など） |
| `organizations/{id}/menus` | 支援メニュー（matchingCount・isVisible・sort_order など） |
| `organizations/{id}/menus/{id}/matchings` | 支援成立記録（成立ごとに自動 ID・継続有無・完了日時） |
| `posts` | 活動日記投稿 |
| `inquiries` | 支援申請（ステータス・lastReminderSentAt など） |
| `categories` | カテゴリマスター（Firestore から動的読み込み） |
| `admins` | 管理者一覧（uid・email） |

## 🔐 セキュリティ

| 項目 | 内容 |
|------|------|
| 認証 | Firebase Authentication（メール / パスワード） |
| 権限管理 | Firebase Custom Claims（`admin: true` で管理者判定） |
| Firestore ルール | 役割ごとに読み書きフィールドを hasOnly で厳密に制限 |
| メール確認 | 登録後にメール確認リンクを送付し未確認は一部機能制限 |
| レート制限 | Netlify Functions にてインメモリで IP 別 5 回 / 60 秒制限 |
| XSS 対策 | 全動的値を `esc()` でサニタイズしてから innerHTML に挿入 |
| ヘッダーインジェクション対策 | メール送信前に改行文字を除去するサニタイズ処理 |
